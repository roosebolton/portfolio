## HTML CSS JS DOM ##
### Beschrijving werkingsgebied html, css, javascript, ajax mbt webapplicaties in eigen woorden ###
Een webapplicatie biedt een user de mogelijkheid via een webpagina een programma te gebruiken. Zo'n
webpagina is dan de grafische interface. Webpaginas worden doorgaan aan de gebruiker getoond via een browser. 

### DOM ###
De DOM is een boomstructuur voor html, die door de browser wordt geparsed. Dit gebeurt van boven naar beneden. Alle knopen in de boomstrucuur bestaan uit betekenisvolle tags, waarvan de html tag de root is in de DOM structuur.

### HTML ###
HTML is applicatielaag taal die bestaat uit betekenisvolle tags. Aan de hand van de betekenis van de tags, de volgorde en nesting, weet de browser wat er moet worden weergegeven. 
Tags kunnen worden verrijkt met extra informatie, die de weergave van de webpagina kunnen beinvloeden, of andere talen, zoals css en javascript, kunnen helpen bij het toekennen van uiterlijke kenmerken of gedrag aan de bewuste html elementen. Er kunnen bijvoorbeeld klassenamen of identiteiten aan worden toegeken, urls aan worden toegevoegd, of andere voor de specifieke tag relevante en parsable informatie.

### CSS ###
Met CSS kan het uiterlijk, de positionering en zelf het gedrag van HTML elementen worden be√Ønvloed. CSS kan html elementen vinden aan de hand van specifieke identifiers, zoals de id of class. Maar er zijn meer identifiers. Het geidentificeerde element kan met CSS allerlei transformaties ondergaan. Css is een krachtig middel om een webpagina een eigen uiterlijk mee te geven. 

### Javascript ###
Javascript is een programmeertaal. Javascript wordt met name gebruikt om front end logica te schrijven, html elementen te manipuleren of creeren, maar zeker ook om met de backend te communiceren via (asynchrone calls).

    ///creates random integer from inclusive to inclusive
    function getRandomInt(min, max) {       
      // Create byte array and fill with 1 random number
      var byteArray = new Uint8Array(1);
      window.crypto.getRandomValues(byteArray);

      var range = max - min + 1;
      var max_range = 256;
      if (byteArray[0] >= Math.floor(max_range / range) * range)
          return getRandomInt(min, max);
      return min + (byteArray[0] % range);
    }

    function firstCharSensitive(length, firstArray, firstArrayLength, nonFirstArray, nonFirstArrayLength){
        output = [];
        if(!propperlength(length)){
          return "Password must be at least 8 characters long";
        }
        else{
          rand1= getRandomInt(0, firstArrayLength-1);
          output[0] = firstArray[rand1];
          for(let i = 1; i < length; i++){
            rand2 = getRandomInt(0, nonFirstArrayLength);
            output[i] = nonFirstArray[rand2];
        }
          return output.join("");
        }
    }
